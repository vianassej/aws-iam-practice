# IAM User Creation Lab
## Objetivo
Criar um usuário IAM com permissões básicas.

## Passos
1. Acesse o console da AWS.
2. Crie um novo usuário.
3. Atribua a política `AmazonS3ReadOnlyAccess`.
4. Teste login com esse usuário.

## Aprendizados
- Diferença entre usuários e grupos.
- Importância de políticas granulares.
